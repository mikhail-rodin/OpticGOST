OpticGOST v1.1

В этой версии OpticGOST:
1. Надстройка для Excel - OpticGOST v1.1. 
Функции:
-строит таблицу конструктивных параметров (r-n-d-диаметры-стрелки) по файлу Zemax Prescription Data
-строит таблицу параметров опт. деталей ОС по тому же файлу
-строит таблицу хода лучей по тексту, сохранённому из окна Analysis/Raytrace
2. Макрос Analysis_export для Zemax
Экспортирует из Zemax графики аберраций в картинках .bmp и в протабулированном текстовом виде.
Картинки и текстовые файлы сохраняются рядом с файлом .zmx

УСТАНОВКА
I. Установка надстройки для Excel.
	1. В параметрах Excel разрешите выполнение надстроек без цифровой подписи:
		Параметры -> Центр управления безопасностью -> Параметры центра управления безопасностью
			В окне "Центр управления безопасностью":
			Надстройки -> выключить "Все надстройки приложений должны быть подписаны надёжным издателем"
	2. Из параметров Excel откройте окно "Надстройки"
		Параметры -> Надстройки -> Управление (внизу)
		Выберите "Надстройки Excel" -> Перейти 
	3. В окне "Надстройки" нажмите "Обзор" и перейдите в папку OpticGOST/OpticGOST_for_Excel
		Выберите OpticGOSTv1.1.xlam

II. Установка макроса для Zemax
	Запустите "Установка макроса для Zemax.bat" 
		При этом скрипт install_set_paths.ps1 создаст файл в папке OpticGOST файл analysis_export.zpl
		и скопирует его в папку /Documents/Zemax/Macros.
		В меню Macros в Zemax должен появиться макрос ANALYSIS_EXPORT
		
		NB! Если вы переместили папку OpticGOST, макрос надо будет переустановить,
		так как сгенерированный скриптом файл analysis_export.zpl содержит ссылки на папку
		OpticGOST/config.
